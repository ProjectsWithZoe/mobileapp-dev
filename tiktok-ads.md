## Filming Notes

- Record screen on a MacBook (dark mode preferred — matches the app's dark theme)
- Use QuickTime or Loom for screen capture
- Add captions via TikTok's built-in text tool or CapCut
- Background music: lo-fi or light electronic (TikTok trending sounds)
- Keep face-cam optional — screen recordings perform well for dev tools
